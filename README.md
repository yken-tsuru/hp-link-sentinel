# Link Sentinel - リンク切れチェッカー

Webサイトを再帰的にクロールし、リンク切れを検出するモダンなWebアプリケーションです。

## 特徴
- **再帰的クロール**: 指定したURLからサイト内を巡回し、すべてのページをチェックします。
- **深さ制限 (Depth)**: クロールする階層の深さを指定して、無限ループや過度なクロールを防ぎます（デフォルト: 3）。
- **ドメイン制御**: 起点URL以外のドメイン（サブドメインや関連サイトなど）も許可リストに追加してクロール対象に含めることができます。
- **リアルタイム検知**: リンク切れが見つかると即座に画面に表示されます。
- **美しいUI**: 直感的でモダンなインターフェース。
- **外部リンクチェック**: 外部サイトへのリンクもチェック（HEADリクエストで負荷を軽減）。

## インストールと実行

1. **依存関係のインストール**
   ```bash
   npm install
   ```

2. **アプリの起動**
   ```bash
   npm start
   ```

3. **ブラウザでアクセス**
   [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方
1. **URL**: チェックしたいWebサイトのURLを入力します（必須）。
2. **深さ (オプション)**: リンクを辿る階層の深さを指定します。1を指定すると入力したページのみチェックします。
3. **許可ドメイン (オプション)**: 追加でクロールしたいドメインがあればカンマ区切りで入力します（例: `blog.example.com, shop.example.com`）。
4. 「スキャン開始」ボタンを押します。
5. ログと結果がリアルタイムで表示されます。
6. いずれかのタイミングで停止したい場合は、「停止」ボタンをクリックしてください。

## 注意事項
- このツールは再帰的にリンクを辿ります。対象サイトに負荷をかける可能性があるため、自身の管理するサイトまたは許可を得たサイトでのみ使用してください。
- 外部サイトへのリンクチェックは、サーバー設定によってはブロックされることがあります（403 Forbiddenなど）。
